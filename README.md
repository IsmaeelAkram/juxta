Juxta is a high-performance, easy-to-use Discord bot.

## Planned features
- Server setup (guides you through the process of making a server)
- Moderation
- Fun
- Music
- Levels/XP
- Money (maybe)

## Juxta uses:
- discord.py
- Redis
- flask
